# Roadmap

Version 1.0 is a complete local Codex status accessory: lifecycle state, safe approvals, multi-session arbitration, notch and non-notch presentation, optional Caps Lock signaling, diagnostics, onboarding, updates, and distribution.

Post-1.0 maintenance will track official Codex hook schemas, reproducible accessibility/display reports, localization, and stable first-party routing APIs if they emerge. Accounts, cloud sync, remote approvals, transcript parsing, other agents, token dashboards, terminal routing, analytics, ads, and novelty themes remain outside scope.
